# 🌐 Premium CV Website

A modern, fully responsive **CV / Resume website** built using **HTML, CSS, and JavaScript** with a clean UI and glassmorphism-inspired design.

---

## 🚀 Features

* 💎 Modern UI design (clean + professional)
* 📱 Fully responsive (mobile-friendly)
* 🎨 Glassmorphism-style components
* 📊 Skill progress bars with animation
* 🌍 Languages section with visual indicators
* 💼 Detailed experience & projects
* 🎓 Full academic CV structure (multi-section)
* 🖨️ Print-friendly layout (A4 support)
* ⚡ Smooth animations using JavaScript

---

## 📁 Project Structure

```
cv-website/
│── index.html      # Main CV structure
│── style.css       # Styling & layout
│── script.js       # Animations & interactions
```

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Glassmorphism)
* JavaScript (DOM + animations)

## ⚙️ How to Use

1. Clone the repository:

   ```bash
   git clone https://github.com/probal2005/CV.git
   ```

2. Open the project folder:

   ```bash
   cd cv-website
   ```

3. Run the project:

   * Open `index.html` in your browser

---

## 🖨️ Print as PDF

* Open the website in browser
* Press **Ctrl + P**
* Select **Save as PDF**
* Choose **A4 size**

---

## 🎯 Customization

You can easily customize:

* ✏️ Text content (edit `index.html`)
* 🎨 Colors & UI (edit `style.css`)
* ⚡ Animations (edit `script.js`)
* 🖼️ Profile image (replace image URL)
